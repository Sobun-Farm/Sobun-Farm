# Sobun-Farm
2024 데이터베이스프로그래밍 1팀 "소분팜" 레포지토리

<br>

### Java 기반의 웹 사이트 : 🌱 소분팜 🌱
- 데이터베이스프로그래밍 과목 최종 프로젝트
- 자취생과 1인 가구를 위한 물품 소분 및 지역 커뮤니티 플랫폼
- 팀원: 김나희, 류지우, 이가연, 장하연
  <br><br>

### 1. 🖥️ 주요 기능 🖥️
- 회원가입, 로그인, 로그아웃 기능
- 소분 물품 등록 기능
- 소분 거래 참여/완료 기능
- 소분 거래 채팅 기능
- 마이페이지 기능
