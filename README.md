## 프로젝트 소개 : 소분팜
소분팜: 소분으로 함께하는 새로운 거래 경험

### 1. 서비스 소개
<img width="100%" src="https://github.com/user-attachments/assets/21107581-1bd3-49c6-a684-e42f57658de0"/><br>
<소분팜>은 자취생과 1인 가구를 대상으로, 선호 지역 내 사용자들과 물품을 소분하여 경제적 이득을 얻고 지역 커뮤니티를 활성화하는 플랫폼입니다.
